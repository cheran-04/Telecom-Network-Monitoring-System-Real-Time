# Telecom-Network-Monitoring-System-Real-Time
Tech Stack: FastAPI, WebSockets, PostgreSQL, Streamlit, Docker Compose
• Built a real-time telecom KPI monitoring system with FastAPI, WebSockets, and PostgreSQL to track latency, jitter,
bandwidth, and packet loss across network towers.
• Developed a live Streamlit dashboard with WebSocket-based data streaming for instant updates and alert visualization.
• Containerized backend, frontend, and database using Docker Compose, enabling reproducible multi-service deployment.
